# Middleclick Discord
Middle click on gif will redirect to your link.

## Installation
```shell
git clone https://github.com/Mon4ik/middleclick-discord.git
cd middleclick-discord

npm install

node ./index.js # runs on 666 port 
```

## Config
Config file available at [config.json](config.json)

## How works?
- When discord makes request, server sends gif
- But when NOT discord, server redirecting
